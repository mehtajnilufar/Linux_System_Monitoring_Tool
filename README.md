# Linux System Monitoring Tool

This project is a Python-based system monitoring utility that tracks CPU, RAM, and Disk usage in real time. It logs the system health into a log file and warns when usage crosses defined thresholds.

## ✅ Features
- Monitors CPU, RAM, and Disk usage every 5 seconds
- Logs system usage and warnings to `log.txt`
- Threshold-based alerts:
  - CPU > 80%
  - RAM > 80%
  - Disk > 90%
- Fully automated, runs continuously
- Useful for DevOps, Cloud, and System Monitoring

## ✅ Tech Stack
- Python
- psutil library
